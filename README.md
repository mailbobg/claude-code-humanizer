# Plain — Claude Code output style

让 Claude Code 说人话的 output style。

- [中文说明](README_cn.md)
- [English guide](README_en.md)

## Quick Install

```bash
chmod +x install.sh && ./install.sh
```

This installs `plain.md` into `~/.claude/output-styles/` and sets it as the global default Claude Code output style.
